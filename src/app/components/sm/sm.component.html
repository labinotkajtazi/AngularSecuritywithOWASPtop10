-Security Missconfiguration-

Just like any other framework Angular has its own security configuration options. Certain ones if not used properly may cause unwanted security behaviour.


-A list of some examples-

bypassSecurityTrustHtml(value: string)
-Inform Angular to trust whatever HTML code it is processing
bypassSecurityTrustScript(value: string)
-Inform Angular to trust whatever Script code it is processing
bypassSecurityTrustStyle(value: string)
-Inform Angular to trust whatever CSS code it is processing
bypassSecurityTrustUrl(value: string)
-Inform Angular to trust the URL it takes as a parameter.May be used to load code from <hyperlinks> or <img src>
bypassSecurityTrustResourceUrl(value: string)
-Inform Angular to trust the URL it takes as a parameter.May be used to load code from <script src> or <iframe src>
