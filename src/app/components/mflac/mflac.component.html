-Missing Function Level Access Control-

Applications are vulnerable to this type of security threat when unauthorized users can make requests/access parts of the web Application they are not supposed to. In order to prevent such attacks, developers must make sure unauthorized users do not have access to restricted parts through UI or directly from URL. Angular provides protection to this type of attack through its routing and authguard modules, but its implementation must be made by the developer, and further checks for authorization must also be made on the back end.


-Routing Code used to prevent unauthorized requests-

 Routes = [
        { path: '', redirectTo: '/home', pathMatch: 'full' },
        { path: 'home', component: HomeComponent },
        { path: 'login', component: LoginComponent, canActivate: [RegisterLoginGuard] }];
