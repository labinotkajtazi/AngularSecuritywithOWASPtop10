-Sensitive Data Exposure-

Sensitive Data Exposure occurs when certain sensitive data is saved in local storage, therefore leaving a security hole which enables other users to have access to that information One way to do this is to store data in the javascript local storage, which would save said data locally permanently unless removed manually.


-The Code responsible for such behaviour-

    Saving to local storage:
        public lStorageVar: string = "";
        localStorage.setItem('localVar', JSON.stringify(this.lStorageVar));
        this.lStorageVar = JSON.parse(localStorage.getItem('localVar'));
        

    Clearing local storage
        localStorage.clear();
        this.lStorageVar = JSON.parse(localStorage.getItem('localVar'));
